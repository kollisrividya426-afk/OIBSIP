# Retail Analysis Project - Oasis Infobyte (Level 1)

## 📌 Project Overview
This project is part of the Data Analytics Internship at Oasis Infobyte (OIBSIP). The goal is to analyze retail sales data, uncover business insights, and build predictive models using Python.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn

## 🚀 Key Steps Performed
1. **Data Cleaning:** Handled missing values and formatted data types.
2. **Exploratory Data Analysis (EDA):** Analyzed top-selling products, sales trends over time, and regional performance.
3. **Visualizations:** Created insightful charts and graphs to represent sales patterns.
